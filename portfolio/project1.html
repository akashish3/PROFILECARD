Count Character



Discription:

Character Counter and Dark Mode Toggle This project provides a simple web interface
for counting characters and words in a text input, selecting characters, and toggling
between light and dark modes.

Features:

Character Count: Displays the number of characters in the input text.

Word Count: Displays the number of words in the input text.

Selected Character Count: Displays the number of selected characters in the input text.

Warning: Shows a warning message if the character count exceeds 200.

Warning:Dark Mode Toggle: Allows users to switch between light and dark modes.
